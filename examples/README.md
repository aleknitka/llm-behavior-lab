# Examples

Each example lives in its own directory with:

- a `README.md` containing prerequisites and launch instructions;
- a version-controlled marimo notebook;
- reusable validated configuration files beside the notebook.

Examples preview and validate their study design before any provider call. Expensive
or artifact-producing cells must be guarded by an explicit user action.

## Available Examples

- [`ollama-bfi10-factorial`](ollama-bfi10-factorial/README.md): run a 300-person
  BFI-10 factorial experiment against a local Ollama model.
